# Data Structures and Algorithms - II Alternate Case Studies

## 📚 Project Overview

This repository contains a collection of alternate real-world case studies for **Data Structures and Algorithms - II**. The case studies are designed to strengthen understanding of advanced data structures and graph algorithms through practical applications and analytical problem-solving.

The project covers:

* Binary Search Trees (BST) and AVL Trees
* Segment Trees and Fenwick Trees
* Minimum Spanning Trees (Kruskal's and Prim's Algorithms)

Each case study follows a university examination pattern and includes:

* Industry-inspired scenarios
* Algorithm construction and analysis
* Tree and graph visualizations
* Step-by-step solution keys
* Java-style pseudocode
* Complexity and performance discussions

## 📖 Case Study 1: BST and AVL Tree Indexing

### Scenario

An e-commerce platform maintains an in-memory product search index to ensure fast product-ID lookups during flash sales.

### Key Concepts

* Binary Search Tree Construction
* AVL Tree Rebalancing
* LL, RR, LR, RL Rotations
* Search Performance Analysis
* Tree Height Evaluation
* AVL Deletion Operations

### Learning Outcomes

* Construct BST and AVL Trees
* Analyze search efficiency
* Implement AVL balancing
* Maintain logarithmic lookup performance

---

## 📖 Case Study 2: Segment Tree and Fenwick Tree Range Queries

### Scenario

A smart hostel energy monitoring system records electricity usage across multiple floors and requires efficient range-sum operations and updates.

### Key Concepts

* Segment Tree Construction
* Range Sum Queries
* Point Updates
* Query Optimization
* Fenwick Tree Comparison
* Complexity Analysis

### Learning Outcomes

* Build and analyze Segment Trees
* Execute range queries efficiently
* Handle dynamic updates
* Compare Segment Tree and Fenwick Tree approaches

---

## 📖 Case Study 3: Minimum Spanning Tree for IoT Gateway Design

### Scenario

A smart campus aims to connect multiple IoT gateway rooms using wired backbone links while minimizing installation cost.

### Key Concepts

* Weighted Graph Representation
* Kruskal's Algorithm
* Union-Find (Disjoint Set Union)
* Cycle Detection
* MST Construction
* Network Cost Optimization

### Learning Outcomes

* Apply MST algorithms
* Use Union-Find for cycle detection
* Optimize network connectivity
* Analyze graph-based infrastructure design
